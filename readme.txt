Integrate WooCommerce with Olivery.

== Description ==

Enable you to integrate WooCommerce with Olivery so you can send a copy of the order to Olivery directly from your admin dashboard.
this will enable Olivery to proccess your order easlly .
this plugin is usefull for those who using Olivery and allready have a Wordpress-Woocommerce store 


== Frequently Asked Questions ==

= Who can use this plugin ? =

This plugin is usefull for those who using Olivery and allready have a Wordpress-Woocommerce store .

= How can i use this plugin ? =

After you install and activate the plugin you will see a new menu link named 'Olivery', this is where you can fill your settings that you can get from Olivery Support. beside that in your order edit admin page you will be able to see a button named 'send to olivery' , click on it and a copy of your order basic informatin will be sent to Olivery in order to proccess your order , you will also be able to see a new choices in orders bulk actions to do the same thing


= Do i need woocommerce? =

Yes,sure , this plugin have no meaning without woocommerce.

= What is Olivery platform? =

Olivery is one stop shop for all logistics solutions, we use the top technologies to enhance your logistics and financial operations. We believe logistics could do more than just a delivery.
